An interactive, fun, space themed Pinball Machine where you score "planets" into stars! <br>

How to Play: <br>
First, use the yellow button on the breadboard to start the game. The scoreboard will turn on and a planet will be released. USe the flippers to score the planets into any of the stars on the playfield to gain a point and release another planet. If you lose your planet, or fail to score a point in 30 seconds, the game will end and your final score will be displayed. <br>
<br>
Quick Start: <br>
-Open the circuit model link above <br>
-Change the distance of the ultrasonic sensor to under and over 5cm <br>
-View the changes in score occurring <br>
<br>
Features: <br>
-Point Scoring with an ultrasonic sensor <br>
-Score tracking with an LCD display <br>
-Auto-Ball release with a servo motor <br>
<br>
How it Works: <br>
-The machine is run with an Arduino Uno Microcontroller programmed in C++ <br>
-The machine uses an ultrasonic sensor to detect when a point is scored so points can be added and a new ball can be released <br>
